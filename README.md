# StartStory
Novel game.

## Git Commands
### Clone repository
```
$ git clone https://github.com/Arfikait-Sirius/StartStory.git
```

### Check branches
```
$ git branch
```

### Change branch
```
$ git switch [branch name]
```

### Merge remote branch into local branch
```
$ git pull origin [remote branch name]
```

### Create branch
```
$ git switch -c [new branch name]
```
